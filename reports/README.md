# Reports Directory

Generated reports and artifacts organized by service.

## Structure

```
reports/
├── trading/                    # Opening Range strategy reports
│   ├── figures/                # Generated charts and visualizations
│   │   ├── audit/                 # Data audit charts
│   │   ├── backtest/              # Equity curves, trade distributions
│   │   ├── risk/                  # Monte Carlo, drawdown charts
│   │   └── robustness/            # Parameter sweep heatmaps
│   └── tables/                 # CSV data exports
│       ├── audit/                 # Data quality summaries
│       ├── backtest/              # Trade logs, daily P&L
│       ├── risk/                  # Risk metrics, regime analysis
│       └── robustness/            # Sensitivity tables
└── govtrades/                  # (Future) GovTrades reports
```

## Trading Reports

### Figures

- `audit/` - Data quality visualizations, gap analysis
- `backtest/` - Equity curves, win/loss distributions
- `risk/` - Monte Carlo simulations, drawdown analysis
- `robustness/` - Parameter sensitivity heatmaps

### Tables

- `audit/` - `data_audit_summary.csv`, `valid_days.csv`
- `backtest/` - `backtest_daily.csv`, `backtest_summary.csv`
- `risk/` - `mc_trade_shuffle.csv`, `perf_summary.csv`
- `robustness/` - `sl_tp_sweep.csv`, `zone_sweep.csv`

## Generation

Reports are generated by running notebooks in `notebooks/trading/`.
