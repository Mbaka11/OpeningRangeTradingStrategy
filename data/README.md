# Data Directory

This directory contains data files organized by service.

## Structure

```
data/
├── trading/              # Opening Range trading data
│   ├── historical/       # Historical M1 OHLCV data (backtesting)
│   │   └── DAT_ASCII_NSXUSD_M1_*.csv
│   └── replay/           # Replay files for paper trading simulation
│       └── replay_YYYY-MM-DD.csv
└── govtrades/            # (Future) GovTrades data cache
```

## Trading Data

### Historical (`trading/historical/`)

- Source: HistData.com or similar
- Format: Semicolon-delimited CSV
- Columns: `datetime;open;high;low;close;volume`
- Used for: Backtesting, parameter optimization

### Replay (`trading/replay/`)

- Generated by: `fetch_session.py`
- Format: Same as historical
- Used for: Paper trading simulation, testing without live market
